
# Book Price Finder (Skeleton)

This bundle matches the structure we discussed. Scraper implementations are minimal
placeholders; you can extend them using your saved HTML samples for Amazon, eBay,
AbeBooks, Bookshop.org, and APPI.org.

Usage:

```bash
pip install -r requirements.txt
python book_price_finder.py your_books.txt
python gui.py  # optional GUI
```
